
## Stock Market Dashboard Using Alpha Vantage API

### Getting Started

#### Prerequisites
- Node.js installed.

#### Installation
1. Clone the repository:
   ```
   git clone https://github.com/bsingh6636/Stock_Market
   ```
2. Navigate to the project directory:
   ```
   cd Stock-Market
   ```
3. Install necessary packages:
   ```
   npm install
   ```
4. Start the project:
   ```
   npm run start
   ```

### Overview
- Animated stock price charts.
- Sector performance comparisons.
- Market sentiment indicators.
- Popular U.S. stock updates.
- Stock search functionality.

### Features
- Animated Charts with Chart.js and react-chartjs-2.
- Sector Performance Analysis.
- Market Sentiment Indicator.
- Latest Stock Updates.
- Search Functionality.
- User Authentication with Google Firebase.
- Dark and Light Mode with Redux.

### APIs Used
- Alpha Vantage API.
- Financial Modeling Prep API.

### Libraries and Tools
- Chart.js, react-chartjs-2.
- chartjs-adapter-moment.
- Redux.
- Google Firebase.
